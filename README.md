# CubeMart Payment Service

![CubeMart](https://img.shields.io/badge/CubeMart-Payment-0f766e)
![Node.js](https://img.shields.io/badge/Node.js-Service-339933)
![gRPC](https://img.shields.io/badge/gRPC-Payments-1f2937)

Payment microservice responsible for validating and processing checkout payment requests in CubeMart.

## Responsibilities

- Accept payment charge requests
- Validate card details
- Return payment transaction responses
- Expose gRPC health checks
